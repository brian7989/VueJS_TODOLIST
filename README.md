# VueJS_TODOLIST
Simple TODO list implementation using VueJS

<a href="https://media.giphy.com/media/3ohzgD1wRxpvpkDCSI/giphy.gif"><img src="https://media.giphy.com/media/3ohzgD1wRxpvpkDCSI/giphy.gif"/></a>

**Main Concepts**

* Data Biding (One Way & Two Ways)
* Event Handling
* Computed Properties
* Methods
* V-for (Looping)
* V-if (Conditional)
* Nesting Components
* Data Transfer via Props & Events ($emit)
* Life Cycle Hooks (Created())
